### 指针的概念

指针是存储另一个变量的内存地址的变量。go语言的指针不能进行偏移和运算，只需要两个符号：`&`取地址，`*`根据地址取值。

![指针](https://cdn.learnku.com/uploads/images/201805/12/23/u01PwjVvCw.png?imageView2/2/w/1240/h/0)

> 空指针

当一个指针被定义之后没有分配任何变量时，它的值为`nil`。

### 指针的声明
```go
var name *Type

type Aminal strcut {
	Hand    int
	Foot    int
}

a := &Aminal{}

var b *Aminal
```
